# 📊 Employee Management System – Power BI Dashboard

## 📌 Project Overview

This project is a **Power BI dashboard (.pbix)** built to analyze and visualize employee data in an organization. It provides meaningful insights into employee distribution, salary structures, payroll trends, and workforce patterns to support data-driven decision-making.

---

## 🎯 Objectives

* To analyze employee data efficiently using interactive dashboards
* To understand salary distribution across departments
* To identify high-performing and highly paid employees
* To track payroll and leave trends over time
* To provide actionable insights for HR and management

---

## 📂 File Included

* `inno pjt.pbix` – Power BI dashboard file containing data model, visualizations, and reports

---

## 📊 Key Features / Analysis

### 🔹 Employee Insights

* Employee distribution across departments
* Qualification-based employee analysis

### 🔹 Salary Analysis

* Average salary per department
* Highest paid employees
* Department-wise salary expenditure

### 🔹 Payroll & Trends

* Payroll trends over time
* Monthly/Yearly salary patterns

### 🔹 Leave Analysis

* Leave patterns among employees
* Department-wise leave trends

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* Data Modeling
* DAX (Data Analysis Expressions)
* Data Visualization Techniques

---

## 📈 Dashboard Highlights

* Interactive filters and slicers
* Dynamic charts and KPIs
* Clean and user-friendly UI
* Drill-down capabilities for deeper insights

---

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Explore dashboards using filters and visuals

---

## 💡 Key Insights (Example)

* Certain departments have higher salary expenditures
* Payroll trends show growth over time
* Leave patterns indicate peak absence periods
* Salary distribution varies significantly across roles
